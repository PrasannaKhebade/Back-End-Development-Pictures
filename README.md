# Back-end Application Development Capstone Project

This repository contains the **Back-end Application Development Capstone Project**, developed as part of Final Project of the IBM Back-End Development Professional Certificate.

It demonstrates the design and implementation of a backend system using modern development practices including REST APIs, microservices, and database integration.

---

## Project Overview

This project simulates a real-world backend application with multiple services working together.

Key objectives:
- Build RESTful APIs
- Integrate backend services with a web framework
- Perform CRUD operations
- Work with NoSQL databases
- Deploy and test backend applications

---

## Tech Stack

- **Languages:** Python, JavaScript  
- **Frameworks:** Flask, Django  
- **Database:** MongoDB  
- **Tools:** Docker, Kubernetes, OpenShift  
- **Concepts:** REST APIs, Microservices Architecture  


---

## Features

- REST API development using Flask  
- Django-based web application integration  
- CRUD operations for backend resources  
- MongoDB database connectivity  
- Microservices-based architecture  
- Containerization and deployment support  

---

## Core Functionalities

### Flask Services
- API endpoints for backend operations  
- Data handling and processing  
- Database interaction  

### Django Application
- Handles web interface and routing  
- Integrates with backend APIs  
- Manages models and migrations  

### Deployment
- Docker-based containerization  
- Scalable deployment-ready setup
